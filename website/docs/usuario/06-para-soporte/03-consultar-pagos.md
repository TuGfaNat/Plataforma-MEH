### Consultar Pagos

#### ¿Para qué sirve?
Permite revisar el estado de un pago específico a solicitud del usuario.

#### ¿Quién puede usarlo?
| Rol | ¿Puede usar? |
|---|---|
| ADMIN | ✅ |
| ORGANIZADOR | ❌ |
| MODERADOR | ❌ |
| SOPORTE | ✅ |
| EMBAJADOR | ❌ |
| MIEMBRO | ❌ |

#### Requisitos previos
- [ ] Permisos de soporte

#### Paso a paso

1. **Accede a la pantalla:** Ingresa a `Consultar Pagos` desde el menú lateral.
2. **Interactúa con la interfaz:** Completa los campos o selecciona las opciones requeridas.
3. **Confirma la acción:** Haz clic en el botón principal para confirmar.
