### Ver Pedidos

#### ¿Para qué sirve?
Visualiza y gestiona el estado de las compras de souvenirs.

#### ¿Quién puede usarlo?
| Rol | ¿Puede usar? |
|---|---|
| ADMIN | ✅ |
| ORGANIZADOR | ❌ |
| MODERADOR | ❌ |
| SOPORTE | ❌ |
| EMBAJADOR | ❌ |
| MIEMBRO | ❌ |

#### Requisitos previos
- [ ] Tener rol de administrador

#### Paso a paso

1. **Accede a la pantalla:** Ingresa a `Ver Pedidos` desde el menú lateral.
2. **Interactúa con la interfaz:** Completa los campos o selecciona las opciones requeridas.
3. **Confirma la acción:** Haz clic en el botón principal para confirmar.
