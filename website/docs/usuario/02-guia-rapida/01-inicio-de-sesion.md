### Inicio de Sesión

#### ¿Para qué sirve?
Permite iniciar sesión en la plataforma y recuperar la contraseña.

#### ¿Quién puede usarlo?
| Rol | ¿Puede usar? |
|---|---|
| ADMIN | ✅ |
| ORGANIZADOR | ✅ |
| MODERADOR | ✅ |
| SOPORTE | ✅ |
| EMBAJADOR | ✅ |
| MIEMBRO | ✅ |

#### Requisitos previos
- [ ] Tener una cuenta creada

#### Paso a paso

1. **Accede a la pantalla:** Ingresa a `Inicio de Sesión` desde el menú lateral.
2. **Interactúa con la interfaz:** Completa los campos o selecciona las opciones requeridas.
3. **Confirma la acción:** Haz clic en el botón principal para confirmar.
