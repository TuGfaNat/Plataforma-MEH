### Para Miembros

#### ¿Para qué sirve?
Guía de funcionalidades exclusivas para los miembros de la plataforma.

#### ¿Quién puede usarlo?
| Rol | ¿Puede usar? |
|---|---|
| ADMIN | ❌ |
| ORGANIZADOR | ❌ |
| MODERADOR | ❌ |
| SOPORTE | ❌ |
| EMBAJADOR | ❌ |
| MIEMBRO | ✅ |

#### Requisitos previos
- [ ] Tener una sesión iniciada

#### Paso a paso

1. **Accede a la pantalla:** Ingresa a `Para Miembros` desde el menú lateral.
2. **Interactúa con la interfaz:** Completa los campos o selecciona las opciones requeridas.
3. **Confirma la acción:** Haz clic en el botón principal para confirmar.
