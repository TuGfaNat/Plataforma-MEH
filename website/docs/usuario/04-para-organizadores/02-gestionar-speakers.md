### Gestionar Speakers

#### ¿Para qué sirve?
Permite añadir, editar y eliminar conferencistas para un evento.

#### ¿Quién puede usarlo?
| Rol | ¿Puede usar? |
|---|---|
| ADMIN | ✅ |
| ORGANIZADOR | ✅ |
| MODERADOR | ❌ |
| SOPORTE | ❌ |
| EMBAJADOR | ❌ |
| MIEMBRO | ❌ |

#### Requisitos previos
- [ ] Permisos de organización

#### Paso a paso

1. **Accede a la pantalla:** Ingresa a `Gestionar Speakers` desde el menú lateral.
2. **Interactúa con la interfaz:** Completa los campos o selecciona las opciones requeridas.
3. **Confirma la acción:** Haz clic en el botón principal para confirmar.
