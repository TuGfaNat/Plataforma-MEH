### Checkpoints del Evento

#### ¿Para qué sirve?
Configurar y gestionar puntos de control (checkpoints) para asistencia.

#### ¿Quién puede usarlo?
| Rol | ¿Puede usar? |
|---|---|
| ADMIN | ✅ |
| ORGANIZADOR | ✅ |
| MODERADOR | ❌ |
| SOPORTE | ❌ |
| EMBAJADOR | ❌ |
| MIEMBRO | ❌ |

#### Requisitos previos
- [ ] Permisos de organización

#### Paso a paso

1. **Accede a la pantalla:** Ingresa a `Checkpoints del Evento` desde el menú lateral.
2. **Interactúa con la interfaz:** Completa los campos o selecciona las opciones requeridas.
3. **Confirma la acción:** Haz clic en el botón principal para confirmar.
